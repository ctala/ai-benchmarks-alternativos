# Guion de llamada de 15 minutos

La llamada no es para “aprender”. Es para vender y forzar una decisión.

## Minuto 1–4: calificación

Pregunta:

1. ¿Cuántos clientes activos tienen?
2. ¿A cuántos les mandan reportes mensuales?
3. ¿Quién arma esos reportes?
4. ¿Cuánto tiempo les toma al mes?
5. ¿Qué usan ahora? Looker Studio, Sheets, PDFs, manual, AgencyAnalytics, DashThis, etc.
6. ¿Qué es lo más molesto del proceso?

Si no hacen reportes o les toma menos de 2–3h/mes, no son buen prospecto.

## Minuto 5–8: oferta

Dices:

> Por lo que me cuentas, esto sí parece encajar.  
>   
> Lo que hacemos es tomar tus fuentes actuales, armar un reporte white-label para tus clientes y dejarlo listo para que cada mes no tengas que reconstruirlo.  
>   
> Cuesta $200/mes. El primer reporte te lo entregamos en 48–72h. Si no te ahorra al menos 3 horas, te devuelvo el dinero.

## Minuto 9–15: cierre

Pregunta directamente:

> ¿Quieres que empecemos con el primer cliente esta semana?

Si dicen sí:

> Perfecto. Para empezar necesito el pago del primer mes y luego te mando el onboarding. ¿Te paso el link?

Luego silencio.

Si dicen “mándame info”:

> Claro. Pero para entender si esto es prioridad: si el reporte que viste resolviera el problema, ¿pagarías los $200 hoy para empezar?

Si no pueden responder sí, cuenta como no.

# Criterios secundarios de muerte

También abandona o pivota si ves alguno de estos patrones:

### 1. Les duele, pero no $200

Frases típicas:

- “Por $49 sí”
- “Usamos Looker Studio gratis”
- “No pagaría otra suscripción”
- “Esto debería costar $20–$50”

Conclusión: el problema existe, pero el precio/segmento no.

### 2. No es prioridad

Frases típicas:

- “Sí, es molesto, pero no urgente”
- “Lo hace un junior”
- “Ya tenemos una plantilla”
- “Lo resolvemos una vez al mes”

Conclusión: dolor insuficiente.

### 3. Ya está resuelto

Frases típicas:

- “Usamos AgencyAnalytics”
- “Usamos DashThis”
- “Looker Studio nos basta”
- “Nuestro cliente casi no lee los reportes”

Conclusión: mercado saturado o diferenciación débil.

### 4. Quieren servicio custom, no herramienta

Frases típicas:

- “Necesito que alguien me escriba el análisis”
- “Cada cliente tiene un reporte distinto”
- “Lo difícil no es armarlo, es explicar resultados”

Conclusión: quizá hay negocio de servicio/concierge, pero no necesariamente SaaS simple.
