**Plan de Ejecución para Landing Page de AutoFlow AI**

**Objetivo**: Crear una landing page completa para el lanzamiento de *AutoFlow AI*, incluyendo investigación, copy, desarrollo, diseño y revisión.

---

### **Desglose de Tareas y Dependencias**

1. **Investigación de Competencia (researcher - Prioridad: Urgent)**
   - **Tarea**: Investigar 3 competidores directos de herramientas de automatización para startups (ej: Zapier, Make, Tray.io).
   - **Entregable**: Documento con:
     - Características clave de cada competidor.
     - Estrategias de copy (mensajes, CTAs, estructura de pricing).
     - Diseño visual (colores, imágenes, layout).
   - **Contexto**: Necesario para definir diferenciadores y guiar el copy y diseño.
   - **Dependencia**: Ninguna. Puede empezar inmediatamente.

2. **Redacción del Copy (writer - Prioridad: High)**
   - **Tarea**: Escribir el copy para las secciones:
     - **Hero**: Título, subtítulo y CTA principal.
     - **Features**: Beneficios clave con viñetas o descripciones.
     - **Pricing**: Opciones de planes (ej: Básico, Pro, Enterprise).
     - **CTA**: Botones y mensajes para conversión.
   - **Contexto**: Basado en los hallazgos del *researcher* y en los diferenciadores de *AutoFlow AI*.
   - **Dependencia**: Requiere el entregable del *researcher*.

3. **Diseño de Assets Visuales (designer - Prioridad: High)**
   - **Tarea**: Crear:
     - **Imagen Hero**: Banner principal con mensaje claro y diseño atractivo.
     - **Imágenes de Features**: Iconos o ilustraciones para cada característica.
     - **Mockups**: Opcional, si se quiere mostrar la herramienta en acción.
   - **Contexto**: Basado en el copy y en la identidad visual de *AutoFlow AI* (colores, estilo).
   - **Dependencia**: Requiere el copy del *writer* para alinear los mensajes.

4. **Desarrollo del Código (coder - Prioridad: High)**
   - **Tarea**: Generar el código HTML/CSS de la landing page, incluyendo:
     - Estructura responsive (mobile/desktop).
     - Secciones: Hero, Features, Pricing, CTA.
     - Integración de imágenes proporcionadas por el *designer*.
     - Posible uso de frameworks como Bootstrap o Tailwind.
   - **Contexto**: Basado en el copy y los assets visuales.
   - **Dependencia**: Requiere el copy del *writer* y las imágenes del *designer*.

5. **Revisión Final (reviewer - Prioridad: Urgent)**
   - **Tarea**: Revisar:
     - **Calidad del Copy**: Ortografía, claridad, tono y coherencia.
     - **Diseño**: Consistencia visual, usabilidad y atractivo.
     - **Código**: Funcionalidad, responsive design y errores.
     - **Mensaje**: Alineación con los objetivos de *AutoFlow AI*.
   - **Contexto**: Entregables de todos los agentes anteriores.
   - **Dependencia**: Requiere que todas las tareas anteriores estén completas.

---

### **Secuencia de Ejecución**
1. **Fase 1**: Investigación (researcher).
2. **Fase 2**: Copywriting (writer) + Diseño (designer) en paralelo.
3. **Fase 3**: Desarrollo (coder) una vez que el copy y diseño estén listos.
4. **Fase 4**: Revisión (reviewer) y ajustes finales.
