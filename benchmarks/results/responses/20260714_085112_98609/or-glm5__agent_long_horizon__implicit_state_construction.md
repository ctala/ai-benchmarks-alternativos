## Objetivo de la campaña

**Objetivo principal:** Posicionar **Aurora** como el set de cuidado facial premium de referencia en Bogotá y generar 15-20 ventas en el primer mes.

**Objetivos secundarios:**
- Aumentar el reconocimiento de marca para **Aurora**
- Generar al menos 5 testimonios de clientas satisfechas
- Construir una base de contenido evergreen para futuras campañas

---

## Audiencia objetivo

| Característica | Detalle |
|----------------|---------|
| **Ubicación** | Bogotá |
| **Edad** | 28-45 años |
| **Género** | Femenino (principalmente) |
| **Perfil** | Mujeres profesionales con poder adquisitivo medio-alto |
| **Intereses** | Skincare, bienestar, autocuidado, cosmética natural, rutinas de belleza |

| **Dolor** | Quieren resultados visibles, productos de calidad, y están dispuestas a pagar más por una rutina que funcione |
