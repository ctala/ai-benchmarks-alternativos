# 📅 SEMANA 1: Producto Listo

**Meta:** Que todo lo físico esté confirmado y listo para salir al mercado.

## Tareas

### 📦 Stock

- [ ] Tener las 4 variedades de té fabricadas y en mano
- [ ] Contar unidades disponibles para lanzamiento
- [ ] Definir mínimo de stock post-lanzamiento

### 🏷️ Packaging

- [ ] Confirmar que etiquetas/impresión llegaron correctamente
- [ ] Empaque manual de los primeros lotes
- [ ] Verificar diseño en formato físico (colores, tipografía)
- [ ] ¿Faltan insumos? (bolsitas, cajas, cinta, stickers)

### 💰 Precios

- [ ] Calcular costo por unidad (té + packaging + empaque)
- [ ] Definir precio de venta sugerido
- [ ] Definir precio de lanzamiento/early bird (si aplica)
- [ ] Crear tabla de precios mayoristas (si planeas resale)

### 🗂️ Operativa

- [ ] Confirmar lugar de almacenamiento
- [ ] Definir cómo se despachan pedidos
- [ ] Crear sistema simple de seguimiento de inventario

## Entregable
Documento con: X unidades disponibles, costo por unidad, precio de venta, fecha de stock listo.

# 📅 SEMANA 2: Identidad y Packaging

**Meta:** Toda la identidad visual definida y el producto fotografiado para poder vender.

## Tareas

### 🏷️ Naming y marca

- [ ] Confirmar nombre de marca definitivo
- [ ] Verificar disponibilidad en redes sociales y dominio web
- [ ] Registrar nombre/copyright si aplica
- [ ] Definir tagline o frase de marca

### 🎨 Diseño visual

- [ ] Definir paleta de colores (3-4 máximo)
- [ ] Definir tipografía (logo + textos)
- [ ] Diseño final de etiquetas listo para imprenta
- [ ] Crear mockups digitales para redes
- [ ] Confirmar packaging secundario (caja, bolsa, sticker)

### 📸 Fotografía de producto

- [ ] Programar sesión de fotos (profesional o celular + buena luz)
- [ ] Preparar set: mesa, fondos, accesorios (ramitas, tazas, tela lino)
- [ ] Fotos de producto: plano, 45°, lifestyle
- [ ] Fotos de packaging: abierto, cerrado, en contexto
- [ ] Editar y seleccionar 5-8 fotos finales

### 🗣️ Tono de voz y mensaje

- [ ] Definir cómo habla tu marca
- [ ] Escribir bio de Instagram / descripción de tienda
- [ ] Preparar 3-5 frases clave de venta

## Herramientas útiles

| Necesidad | Herramienta |
|-----------|-------------|
| Diseño de etiquetas | Canva, Illustrator |
| Mockups | Smartmockups, Placeit |
| Paleta de colores | Coolors.co |
| Sesión de fotos rápida | Shotroom (app) |

## Entregable
Carpeta con: logo, etiquetas listas para imprenta, 5-8 fotos profesionales, bio, paleta y tipografías documentadas.

- [ ] Serie de posts " conocé los ingredientes/proceso"

- [ ] Enviar muestras a micro-influencers del nicho bienestar/alimentación

- [ ] Engagement activo: responder mensajes, comments, preguntas

# 📅 SEMANA 5: Lanzamiento 🚀

**Meta:** Activate todos los canales simultáneamente y generar la mayor cantidad de ventas posible.

## Tareas

### 🎯 Día 1 (o fin de semana de lanzamiento)

- [ ] Publicación simultánea en todos los canales (IG, web, TikTok)
- [ ] Historia en vivo o video de lanzamiento
- [ ] Comunicación a toda la lista de espera (" YA ESTÁ DISPONIBLE")

### 📲 Contenido de lanzamiento

- [ ] 3-5 posts el día 1 (producto, testimonios, oferta, detrás de cámaras)
- [ ] Stories cada 2-3 horas (countdown final, demos, testimonios)
- [ ] Video de TikTok/Reels mostrando el unboxing

### 💰 Ventas

- [ ] Seguir de cerca los pedidos entrantes
- [ ] Responder consultas en menos de 1 hora
- [ ] Ofrecer upsell (pack de 4 sabores, suscripción, regalo)

### 🎁 Presión de urgencia

- [ ] "¿Solo X unidades disponibles?" (si es real)
- [ ] Contador de tiempo para oferta early bird
- [ ] Testimonios/reviews mostrando primeras entregas

## Entregable
Ventas concretadas desde el día 1, clientes satisfechos con primera experiencia de delivery.
