# 🚀 Plan Maestro de Lanzamiento: Línea de Tés Artesanales (6 Semanas)

*Objetivo: Tener el producto físico listo y los costos definidos.*

3.  **Costos y Precios:** Calcular COGS (Costo de Goods Sold), margen de ganancia (min. 40-50%) y precio final.
4.  **Logística:** Cotizar paquetería, definir tiempos de entrega y preparar cajas de envío resistentes a humedad.
5.  **Plataforma:** Configurar tienda online (Shopify/WooCommerce), cargar SKUs y probar pasarela de pagos.

## **Semana 2: Activos Digitales y Web**
*Objetivo: Crear la experiencia de compra atractiva y funcional.*
1.  **Fotografía:** Sesión de fotos de producto (cerrado, abierto, vapor), estilo de vida y videos cortos (Reels/TikToks).
2.  **Copywriting:** Descripciones con notas de cata, origen y beneficios. Storytelling de marca ("Por qué estos tés").
3.  **Configuración Web:** Subir productos, configurar variantes, instalar pasarelas de pago y políticas (envíos/devoluciones).
4.  **Landing Page:** Crear página de captura de leads para la semana 3 (Oferta de lanzamiento).

## **Semana 3: Generación de Expectativa (Teasers)**
*Objetivo: Crear "hambre" sin vender aún.*
1.  **Redes Sociales:** 3 posts/semana. "Behind the scenes", diseño de empaque, misterio sobre sabores.
2.  **Lista de Espera (Waitlist):** Lanzar Landing Page de captura. Incentivo: 15% descuento + envío gratis.
3.  **Influencers:** Contactar 5-10 microinfluencers del nicho bienestar para envío de muestras.
4.  **Email:** Configurar secuencia de bienvenida para captados.

## **Semana 4: Captación de Leads Intensiva**
*Objetivo: Llenar la base de datos para el lanzamiento.*
1.  **Email Marketing:** Enviar correos a la lista de espera anunciando la fecha exacta.
2.  **Oferta Flash:** "Los primeros 50 pedidos con regalo sorpresa".
3.  **Social Ads (Opcional):** Campaña pequeña de tráfico a la Landing Page de captura.
4.  **Seguimiento Influencers:** Confirmar fechas de publicación de sus contenidos.

## **Semana 5: Semana de Lanzamiento (Ventas)**
*Objetivo: Maximizar ventas en los primeros 7 días.*
1.  **Día 0 (Lanzamiento):** Correo a toda la lista de espera + Post de redes + Stories de "Abierto".
2.  **Día 1-3:** Prueba social (mostrar primeras compras). Recordatorios de oferta limitada.
3.  **Día 4-7:** Cierre de oferta (Urgencia: "Quedan pocos días/stock limitado").
4.  **Soporte:** Atención al cliente rápida (chat/email) para dudas post-compra.
