## **📅 Detalle Exhaustivo: Semanas 1 y 2**

### **Semana 1: Definición y Logística Final**
*Objetivo: Dejar todo listo y operativo para poder empezar a comunicar y vender sin contratiempos.*

#### **Checklist de Acciones Críticas:**

**1. PRODUCTO (Físico)**
- [ ] Aprobación final de muestras (sabores, aroma, presentación).
- [ ] Empaque definitivo (sobres, etiquetas) probado y listo.
- [ ] Certificaciones/etiquetado legal verificado.
- [ ] Stock mínimo de lanzamiento (2-3 semanas de venta) confirmado.

**2. TIENDA ONLINE**
- [ ] Plataforma 100% funcional (productos subidos, precios, variantes).
- [ ] Pasarela de pago configurada y probada.
- [ ] Dominio propio y correo profesional activos.
- [ ] Políticas (envíos, devoluciones, privacidad) publicadas.
- [ ] Checkout optimizado (pasos mínimos, impuestos automáticos).

**3. LOGÍSTICA DE ENVÍOS**
- [ ] Proveedor de envíos definido (tarifas pactadas).
- [ ] Embalaje de envío (cajas, relleno, branding) listo.
- [ ] Proceso de empaque estandarizado.
- [ ] Zonas de cobertura definidas y costos reflejados.

**4. ASPECTOS LEGALES/FINANCIEROS**
- [ ] Facturación configurada (según normativa local).
- [ ] Cuenta bancaria empresarial (recomendado).
- [ ] Presupuesto de lanzamiento cerrado (con margen de seguridad).

#### **❓ Preguntas Clave para tu Contexto (Semana 1):**
1. ¿Tienes ya las **etiquetas físicas** en mano o las mandarás a imprimir esta semana?
2. ¿Ya elegiste la **plataforma de e-commerce** (Shopify, WooCommerce, etc.)?
3. ¿Tienes un **proveedor de envíos** pactado o estás cotizando?
4. ¿Ya calculaste el **precio de venta** (costo + margen + envío)?

### **Semana 2: Creación de Contenido & Materiales**
*Objetivo: Tener toda la identidad visual y el contenido narrativo listo para comunicar tu marca de forma consistente y atractiva.*

#### **Checklist de Acciones Críticas:**

**1. FOTOGRAFÍA PROFESIONAL DEL PRODUCTO**
- [ ] Fotos de producto (estudio): cada sabor, flat lay con elementos de experiencia.
- [ ] Fotos de estilo de vida (lifestyle): personas disfrutando el té en contextos reales.
- [ ] Videos cortos (Reels/TikToks): proceso de preparación, aroma, momento de consumo.

**2. DISEÑO GRÁFICO Y BRANDING**
- [ ] Banners para redes y web (portada, hero image).
- [ ] Graphics para posts (beneficios, infografías de preparación).
- [ ] Kit de prensa/influencers (PDF con historia, fotos en alta, textos listos).

**3. REDACTAR CONTENIDO DE VENTA**
- [ ] Descripciones de producto (notas de cata, historia behind-the-scenes).
- [ ] Historia de la marca (About Us) convincente y auténtica.
- [ ] Copy para redes (calendario de contenido para 4 semanas, hashtags).

**4. PREPARACIÓN DE LISTAS Y HERRAMIENTAS**
- [ ] Lista de contactos inicial (emails, influencers micro).
- [ ] Plantillas de email marketing (preventa, lanzamiento, post-venta).
- [ ] Calendario editorial (Google Sheet/Notion) con fechas y responsables.

#### **⚠️ Errores comunes a evitar:**
- Fotos con baja calidad → Invierte en esto, es lo primero que ve el cliente.
- Textos genéricos → Tu té es *premium*, las descripciones deben reflejar ese valor.
- Dejar el contenido para última hora → La semana 3 es para activar, necesitas material listo.

#### **❓ Preguntas Clave para tu Contexto (Semana 2):**
1. ¿Tienes **presupuesto para fotógrafo** o harás las fotos tú misma?
2. ¿Ya definiste el **tono de voz** de la marca (ej: sofisticado, cercano, místico)?
3. ¿Vas a crear **contenido en video** desde el lanzamiento?
4. ¿Tienes ya una **lista de influencers** en mente para contactar en la semana 3?
