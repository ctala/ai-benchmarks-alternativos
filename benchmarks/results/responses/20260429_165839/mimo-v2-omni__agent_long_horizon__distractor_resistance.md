# 📋 SEMANA 1: Definir Marca y Packaging

## Objetivo
Tener clara la identidad de tu línea y el diseño de sobres/etiquetas para poder producir.

---

### 🔑 Decisiones clave

#### 1. Nombre de la línea
Orientaciones según estilo:
- **Elegante/gourmet**: *Casa Lavanda*, *Colección Botánica*, *Terroir Tea*
- **Moderno/instagrammeable**: *Té & Co.*, *Roots Tea*, *Hoja Santa*
- **Con personalidad**: *Las Teteras Rebeldes*, *Slow Brew*, *Pétalos*

#### 2. Identidad visual
| Elemento | Opciones |
|----------|----------|
| **Paleta de colores** | Cálidos (terracota, dorado, verde oliva) / Frescos (blanco, azul, neutros) / Oscuros (negro, burdeos, dorado) |
| **Estilo** | Minimalista / Ilustrado / Orgánico-artesanal |
| **Tipografía** | Serif elegante / Sans moderna / Handmade |

#### 3. Packaging
- **Formato del sobre**: Kraft, blanco mate, negro premium, con ventana
- **Cierre**: Pegatinos, stickers personalizados, cinta
- **Cantidad**: 2g / 3g / 5g por porción
- **Info obligatoria**: Ingredientes, peso, lote, datos del productor

---

### 📅 Cronograma Semana 1

| Día | Tarea |
|-----|-------|
| **Lunes** | Definir nombre (máx. 3 opciones, elegir 1) |
| **Martes** | Elegir paleta de colores y estilo visual |
| **Miércoles** | Buscar referencias de packaging (Pinterest, Behance) |
| **Jueves** | Redactar copy para etiquetas (nombre sabor, descripción, beneficios) |
| **Viernes** | Diseñar o contratar diseño de etiquetas y sobres |
| **Sábado** | Ajustes finales |
| **Domingo** | Aprobar diseño final → listo para producir |

---

### ✅ Entregables Semana 1

- [ ] Nombre de la línea aprobado
- [ ] Paleta de colores definida (3-4 tonos)
- [ ] Moodboard de referencia visual
- [ ] Copy de las 4 etiquetas redactado
- [ ] Diseño de etiqueta/sobre en borrador o final

---

### 🛠️ Herramientas útiles
- **Moodboard**: Pinterest, Milanote
- **Diseño**: Canva, Adobe Express
- **Colores**: Coolors.co
- **Fuentes**: Google Fonts

# 📋 SEMANA 2: Producir el Lote de Lanzamiento

## Objetivo
Tener tus tés producidos, empaquetados y con fotos profesionales listas para vender.

---

### 🔑 Tareas clave

#### 1. Producción de los 4 sabores
- Definir recetas finales (proporciones exactas)
- Pesar cada sobre para consistencia
- Producir en lotes para mantener calidad uniforme

#### 2. Packaging final
- Imprimir etiquetas y sobres
- Llenar, sellar y etiquetar sobres
- Control de calidad: peso, cierre, apariencia

#### 3. Sesión de fotos
| Tipo de foto | Uso |
|--------------|-----|
| **Producto** | Cada sabor + línea completa (fondo limpio) |
| **Lifestyle** | Tazas de té, ambiente cozy, ingredientes sueltos |
| **Redes** | Formato cuadrado (IG feed) + vertical (Reels/Stories) |

---

### 📅 Cronograma Semana 2

| Día | Tarea |
|-----|-------|
| **Lunes** | Confirmar recetas de los 4 sabores |
| **Martes** | Comprar ingredientes + materiales de empaque |
| **Miércoles** | Producir los tés (blends) |
| **Jueves** | Llenar y sellar sobres |
| **Viernes** | Sesión de fotos |
| **Sábado** | Editar fotos |
| **Domingo** | Revisión final: ¿todo listo para Semana 3? |

---

### ✅ Entregables Semana 2

- [ ] 4 sabores producidos y empaquetados
- [ ] Stock mínimo de lanzamiento (sugerencia: 50-100 sobres por sabor)
- [ ] 10-15 fotos profesionales editadas
- [ ] Carpeta de fotos organizada y lista para usar
